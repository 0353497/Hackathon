# ticket structure 

title - char max 200

description - char max 2000

ticket id - int

employee - int 

client - int

date (unix timestamp) - int

severity/priority - enum

status - enum

catergory - enum

roomID - int (or str if complex like: A1.02)

geocode latlong - double x2

emailTeamLeader - char max 200
